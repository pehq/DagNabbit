AddOn​

</>

offset: Vector2--

Offsetted from the top left of the Matrix

) → ()

Adds a matrix over another matrix. Anything that's a false value won't be overlaid over the base matrix. If the other matrix being added onto the base matrix goes out of bounds then the out of bounds won't be counted. Can be offsetted from the top-left with offset.
